정규식이란?

정규식에서 표준 구문 규칙을 사용하여 문자열 데이터의 간단한
패턴 및 복잡한 패턴을 검색하고 조작할 수 있습니다.

장점 : SQL 및 PL/SQL에서 정규식 사용하여 문자열을 검색하고 조작 가능



오라클 SQL 정규표현식 함수

REGEXP_LIKE
--> Query의 WHERE 절에서 조건을 사용하여 지정한 정규식과 일치하는 행을 반환

REGEXP_REPLACE
-->문자 열에서 패턴을 검색하고 해당 패턴의 각 발생 값을 지정한 패턴으로 대체

REGEXP_INSTR
--> 이 함수는 문자열에서 지정된 정규식 패턴의 발생 값을 검색 

REGEXP_SUBSTR
--> 이 함수는 지정한 정규식 패턴과 일치하는 실제 부분 문자열을 반환

REGEXP_COUNT
--> 이 함수는 입력 문자열에서 패턴 일치가 발견되는 횟수를 반환
